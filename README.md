# 🐍 贪吃蛇 · 多地图版

经典贪吃蛇游戏，包含 4 张不同地图，支持方向键和 WASD 操控，直接在浏览器中畅玩。

## 游戏说明

控制蛇吃食物变长，避开墙壁、障碍物和自己的尾巴。穿墙模式，从屏幕一侧出去会从另一侧回来。

### 操作指南

| 按键 | 功能 |
|---|---|
| **↑ ↓ ← → / WASD** | 控制蛇的移动方向 |
| **空格** | 按住加速 |
| **P** | 暂停 / 继续 |
| **Tab** | 切换地图 |
| **R** | 游戏结束后重新开始 |
| **ESC** | 退出（刷新页面） |

### 四种地图

| 地图 | 描述 |
|---|---|
| 🟢 **经典模式** | 空旷场地，无任何障碍，适合新手练习 |
| 🏗️ **迷宫模式** | 十字迷宫布局，通道狭窄，考验走位 |
| 🟫 **四柱模式** | 四角各有 3×3 的柱子作为障碍 |
| 🔄 **回廊模式** | 环形走廊结构，内外两圈围墙 |

### 规则

- 吃到红色食物 +10 分，蛇身增长一格
- 撞到障碍物或自己的身体 → 游戏结束
- 吃掉所有食物填满地图 → 胜利
- 撞墙会从对面穿出（无边界死亡）

## 在线游玩

👉 **直接打开 [index.html](index.html) 即可在浏览器中运行**（无需安装任何软件）。

也可用 Python 运行桌面版：

```bash
pip install pygame
python snake_game.py
```
